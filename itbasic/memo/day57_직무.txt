2. 보안 직무 
1) 대기업 정보보안팀 
	하는 일 : 기업 내부 정보자산 보호 및 보안 관리 담당 
	기술보안 - 시스템, 네트워크, 어플리케이션 보안 => 침해사고 대응 경험, 모의해킹, 보안 솔루션 운영 
	관리보안 - 보안 정책, 범률 준수 => 개인정보보호법, 정보통신망법, ISMS-P, ISO27001 이해 
	보안 솔루션 운영 & OA 지원 - 사내 보안 솔루션 운영, 사용자 지원

	대기업 공채 OR 경력직 채용 
	신입 경쟁률 높음(서울권 4년제 이상, 정처기, 정보기)

2) 정보보안 전문업체
	기업 고객에게 정보보안 컨설팅, 관제, 침해대응 등의 보안 서비스를 제공 
	
보안관제(SOC)	보안 이벤트 모니터링 및 침해탐지 => SIEM, IDS/IPS, 로그 분석 
		주야 교대 / 내 시간이 많음 
		네트워크 제대로 배울 수 있음 + 엑셀 
CERT(침해사고 대응) 보안 사고 분석 및 대응 => 악성코드 분석, 포렌식, 사고대응 경험 
모의해킹		시스템/네트워크 취약점 분석 및 웹 서비스 해킹 시뮬레이션 
			=> QWASP, Kali Linux, Burp Suite, Metasploit
취약점 진단 	시스템/네트워크 취약점 분석, 보안 위험 분석 및 보안 강화 컨설팅 
			=> 네트워크 및 서버, 웹 보안, CVE 분석
보안시스템구축	신규 보안 솔루션/네트워크/서버장비 구축 및 연동 
보안시스템운영 	고객사 보안 장비/네트워크/서버장비 유지보수 및 운영
